## 아키텍처 요약

```
asset-diary-front (Granite RN + TDS, 독립 프로젝트)
   │  appLogin() → {authorizationCode, referrer}
   ▼
living-craft-backend (NestJS, /api/ad/*)
   │  mTLS → apps-in-toss-api.toss.im/.../oauth2
   │  AES-256-GCM 복호화 → userKey → ad.users upsert → JWT 발급
   ▼
PostgreSQL schema "ad" + @nestjs/schedule cron (정기지출)
```
