T 의 개념은 저번과 같으며


T = 매수누적액 / 1회매수액 (소수점 둘째자리 올림) 입니다.


다만 이번 버전에서는 1회매수액이 실시간으로 변하기 때문에, 이점을 고려하셔야 합니다







별% 의 개념또한 저번과 같습니다. 다만,



이번에는 20분할로 시작하고, TQQQ 는 15% 기준, SOXL 은 20% 기준이기 때문에





< TQQQ 별%지점 = 15 - 1.5T >

< SOXL 별%지점 = 20 - 2T > 가 됩니다


즉 T=10 에서 별%=0 이 되며, 전후반전을 가르는 기준이 됩니다
