일반모드는 소진이 되기전의 쿼터매도를 포함한 일련의 과정을 뜻하며

소진모드는 회차를 뜻하는 T값이 1회분 미만이 남아있는 시점부터 시작해서, 특정 조건을 만족하여 다시 일반모드로 되돌아가기 전의 과정을 뜻합니다

일반모드 방법론을 시작합니다

종목은 초심자는 TQQQ 를 추천하며, 추후 TQQQ 변동성이 익숙해지면 SOXL 투자도 겸해보는 순서로 가세요

수익은 일반적으로 SOXL 무한매수법이 높지만, 처음 무한매수법을 시작한다면 TQQQ 변동성도 못견디고 중간에 포기할 가능성이 높기 때문입니다.

현재 무한매수법 오피셜은 TQQQ , SOXL 이외 다른 종목은 추천하지 않습니다

이 글에서는 이유에 대한 설명은 최대한 배제하고
최대한 간결하게 방법론만 적도록 하겠습니다

---

1. 원금설정 및 분할기간 설정 ( 예전과 동일 )

(1) 무한매수법을 시작하기 전, 원금을 설정합니다. 이 원금에서 매수후 잔금이 남아있더라도 반드시 다른 투자에 사용하지 않습니다. ＜예시로 홍길동씨는 무한매수법을 시행할 원금을 총 2만달러로 시작한다고 설정하겠습니다＞

(2) 분할 정하기 : 20분할/30분할/40분할 정도를 추천합니다. 분할 수가 적을수록 공격적이고 위험하며, 분할 수가 커질 수록 방어적이고 안정적인 성향이 있습니다. ＜홍길동씨가 2만달러로 20분할을 선택한다면, 1회매수금은 1000달러로 시작하게 되며, 40분할을 선택한다면 500달러 매수로 시작하게 됩니다＞

(3) TQQQ와 SOXL 무한매수법을 동시에 할 경우에는, 각 원금을 독립적으로 설정해야합니다. 다만 종목이 다르므로 한 계좌에서 진행해도 괜찮습니다. 만약 TQQQ 무한매수법을 20분할과 40분할을 동시에 한다면, 각자 다른 계좌로 하는 것이 나을 것 같습니다.

---

2. T값의 정의 (이전 버전과 다름)

(1) Turn의 약자로, 진행회차를 의미합니다. 1.0T 는 1회분이 진행된 상태를 의미합니다

(2) 일반모드에서 T는 1회매수시 (+1) / 절반매수시 (+0.5) / 쿼터매도시 (직전T×0.75) 로 예전버전보다 단순화 했습니다. ＜ 예를 들어, T가 7의 값인데, 여기서 1회치가 다 매수되면 T=8 이 되며, 절반치가 매수되면 T=7.5가 되며, 쿼터매도가 발생하면 T=7×0.75=5.25 가 됩니다 ＞

(3) 소수점 몇째자리 등의 제한 없이, 그 숫자 자체로 표현합니다. 예를들면 T 값은 5.165479525 처럼 길어질 수 있습니다

(4) 지정가매도 후 loc 매수되는 경우는, ( ×0.25 + 1 ) or ( ×0.25 +0.5 ) 로 계산합니다

---

3. 별지점의 정의 (예전과 같음)

(1) 별지점은 매수와 매도의 기준을 결정하는 데에 사용되는 값으로, V2.2 부터 새로 생긴 개념입니다

(2) 20분할 TQQQ 별% = ( 15-1.5T ) %
40분할 TQQQ 별%= ( 15-0.75T ) %
20분할 SOXL 별% = ( 20-2T ) %
40분할 SOXL 별% = ( 20-T ) %

(3) 여기서 계산된 별% 값과 평균단가를 곱해서, 별지점을 매일 계산해야 하며
그 별지점을 기준으로 매수와 매도를 가르게 됩니다

예를들면, 20분할 SOXL 무한매수법을 진행중인데, 평단이 38.30달러이고 T=8.6 이라고 가정합니다
별% = ( 20-2×8.6 ) % = 2.8% 이므로
평단 × ( 1+별% ) = 38.30 ×(1+2.8%) = 39.37 $ (반올림) 가 됩니다

이 39.37$ 가 매수와 매도를 가르는 기준이 됩니다

4. 매수금 설정 (이전 버전과 다름)

(1) 처음 매수는 전체 원금에 분할수를 나눈 것으로 시작하지만, 두번째 매수부터는 매수금이 미세하게 변합니다

(2) 20분할 무매 1회매수금 : 잔금 / (20-T)
40분할 무매 1회매수금 : 잔금 / (40-T)

예를들어 원금이 2만달러에 40분할인 경우, 첫 매수시도는 500달러가 됩니다
하지만 실제로 500달러가 모두 사용되는 경우는 없습니다. 478달러가 사용되었다고 가정하겠습니다

그럼 잔금은 20000-478=19522$ 가 되며
회차 T = 1 인 상태이므로

19522 / 39 = 500.5641... $ 가 2회차의 일일매수 시도액 이 됩니다

이는 T가 +1 이든 +0.5 이든 ×0.75이든 상관없이 적용되므로
매일매일 일일매수 시도액이 미세하게 달라집니다
